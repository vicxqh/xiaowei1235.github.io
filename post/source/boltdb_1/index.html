boltdb 简介

是什么

boltdb是纯golang实现的一个键-值数据库，你可以在自己的程序中使用boltdb库创建/操作数据库。boltdb参考LMDB实现，代码简洁也非常适合学习数据库的实现。boltdb的原始库已经归档，coreOS维护的fork用于社区开发者持续迭代。

怎么使用

简单示例如下， 新建一个数据库文件my.db、创建一个新桶MyBucket， 并放入键值对： "my-key", "my-value"。

  // Open the my.db data file in your current directory.
	// It will be created if it doesn't exist.
	db, err := bolt.Open("my.db", 0600, nil)
	if err != nil {
		log.Fatal(err)
	}
	defer db.Close()

	db.Update(func(tx *bolt.Tx) error {
		b, err := tx.CreateBucket([]byte("MyBucket"))
		if err != nil {
			return fmt.Errorf("create bucket: %s", err)
		}
		b.Put([]byte("my-key"), []byte("my-value"))
		return nil
	})

更多的用法可以参考boltdb README 及godoc。

适用场景

  • 程序需要内嵌数据库
  • 只需要简单的键值存储，不需要复杂的SQL查询与处理
  • 数据读多于写
  • 需要事务保证

有哪些项目使用了boltdb?

比如InfluxDB，Consul，Seaweed File System。目前还没有时间探索这些知名项目是如何使用boltdb的。

源码组织结构

只包含重要文件。golang通过编译标志支持多平台可以参考 go build

bolt_os_arch.go   // 含各操作系统、体系架构特殊syscall的实现
bucket.go					// 数据库桶操作及持久化相关代码
cursor.go					// 游标操作相关代码，用于定位读写位置
db.go							// 数据库最顶层抽象
freelist.go				// 空闲页管理逻辑
node.go						// b+树节点相关操作
page.go						// 物理页相关操作
tx.go							// 事务相关操作
